About the Project

Gourmet Vendor Finder is designed to help food enthusiasts discover and review gourmet food vendors in their vicinity. Whether you're looking for a specific cuisine or want to try something new, our platform provides all the necessary details to make your choice easier.

About the Developer

Hi, I'm Onyekelu Chukwuebuka, a passionate developer with a keen interest in creating impactful web applications. I built Gourmet Vendor Finder to combine my love for technology and food, aiming to provide a useful service for food lovers everywhere.

Project Features

  • Find nearby gourmet food vendors
  • Read and write reviews
  • Filter by cuisine and rating
  • User authentication and personalized recommendations

Technical Stack

  • This project is built using React for the frontend,
  • Flask for the backend
  • Postgresql database for data storage
  • The authentication system is implemented with Flask-Login and Flask-Jwt-extended
  • the application is deployed using Vercel.
